Struct RefArg

pub struct RefArg<'r, T> { /* private fields */ }
Expand description

Simple reference wrapper, used when passing arguments by-ref to Godot APIs.

This type is often used as the result of ToGodot::to_godot(), if Self is not a Copy type.
