Expand description

To export properties to godot, you must have an impl-block with the #[godot_api] attribute, even if it is empty.

This trait is automatically implemented when such an impl-block is present. If rust complains that it is not implemented, then you can usually fix this by adding:

ⓘ
#[godot_api]
impl MyClass {}

Where you replace MyClass with the name of your class.
