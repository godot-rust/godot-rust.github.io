Expand description

Enum representing different errors during script instance calls.

Provides a type-safe way to handle call errors when implementing ScriptInstance::call(). It maps to the underlying GDExtensionCallErrorType constants from Godotโs C API.

Note that the OK variant is not included here, as it represents a successful call. This type is meant to be used in Result<T, CallErrorType>, where Ok(T) indicates success.

See SiMut::base_mut() for an example.
