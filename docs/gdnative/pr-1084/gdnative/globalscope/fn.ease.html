Function gdnative::globalscope::ease

pub fn ease(s: f32, curve: f32) -> f32
Expand description

Returns an “eased” value of x based on an easing function defined with curve.

This easing function is based on an exponent. The curve can be any floating-point number, with specific values leading to the following behaviors:

Value rangeEffect
s < -1Ease in-out
s == -1Linear
-1 < s < 0Ease out-in
s == 0Constant
0 < s < 1Ease out
s == 1Linear
s > 1Ease in

See also smoothstep. If you need to perform more advanced transitions, use Tween or AnimationPlayer.
