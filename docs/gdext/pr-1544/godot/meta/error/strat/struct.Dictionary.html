godot::meta::error::strat

Struct Dictionary 

pub struct Dictionary {
    pub error: Variant,
}
Expand description

Error strategy that returns a Dictionary<GString, Variant> – {"ok": val} on success, {"err": err} on failure.

Use this when you want GDScript to receive a result dictionary with a named outcome. The error is converted to a Variant via ToGodot::to_variant(), so any Godot-compatible type can be used as the error.

Implements From<E> for any E: ToGodot, so the ? operator works with any Godot-compatible error type.

§Example

use godot::meta::error::strat;

#[godot_api]
impl MyNode {
    #[func]
    fn try_parse(&self, input: GString) -> Result<i64, strat::Dictionary> {
        input.to_string().parse::<i64>()
            .map_err(|e| strat::Dictionary::new(e.to_string()))
    }
}

GDScript usage:

var result: Dictionary = node.try_parse(input)
if result.has("ok"):
    var value: int = result["ok"]
else:
    print("Parse failed: ", result["err"])
