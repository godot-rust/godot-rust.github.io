Expand description

Trait to create more references from a smart pointer or collection.

Required Methods§

fn share(&self) -> Self

Creates a new reference that points to the same object.

If the referred-to object is reference-counted, this will increment the count.

impl Share for Dictionary

Creates a new reference to the data in this dictionary. Changes to the original dictionary will be reflected in the copy and vice versa.

To create a (mostly) independent copy instead, see Dictionary::duplicate_shallow() and Dictionary::duplicate_deep().
