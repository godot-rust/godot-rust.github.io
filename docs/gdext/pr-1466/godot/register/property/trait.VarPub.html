Expand description

Customizes how #[var(pub)] generates public getter/setter accessors for a property.

#[var] properties only need Var. The pub modifier additionally requires a way to expose the field through generated methods without giving direct field access.

This trait has a blanket implementation for all T: Clone, using clone() in the getter, so in most cases you don’t need to do anything. The generated getters are not designed to provide customization beyond that; Clone will always be the behavior if available. If more special types are required, define getters/setters manually.

§Associated type

Type is the type used by the generated public accessors. It may differ from the stored type (Self) and from the property conversion type (<Self as GodotConvert>::Via). As an example, OnEditor<T> uses Option<T> for Type.
