Attribute Macro gdnative::init::callbacks

#[callbacks]
Expand description

Declares the library-level GDNative callbacks. See [gdnative::init::GDNativeCallbacks].

§Arguments

The attribute takes an optional prefix argument, which can be used to change the prefixes of the symbols generated by this macro. This is used by Godot on platforms where dynamic linking is unavailable. The value of this argument should match the GDNativeLibrary::symbol_prefix property for the library on the Godot side, defaulting to godot_.
