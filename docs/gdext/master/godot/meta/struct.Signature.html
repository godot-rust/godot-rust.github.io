Struct Signature

pub struct Signature<Params, Ret> { /* private fields */ }
Expand description

A full signature for a function.

For in-calls (that is, calls from the Godot engine to Rust code) Params will implement InParamTuple and Ret will implement ToGodot.

For out-calls (that is calls from Rust code to the Godot engine) Params will implement OutParamTuple and Ret will implement FromGodot.
